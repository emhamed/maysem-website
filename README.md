# Maysem Website

A modern, responsive website for Maysem aesthetic clinic in London.

## Structure

- `index.html` - Homepage
- `treatments.html` - Treatments page with all services
- `about.html` - About page
- `styles/` - CSS stylesheets
  - `main.css` - Main styles
  - `navigation.css` - Navigation styles
  - `footer.css` - Footer styles
  - `treatments.css` - Treatments page specific styles
- `scripts/` - JavaScript files
  - `main.js` - Main interactivity and functionality
- `assets/images/` - Image assets folder

## Features

- Responsive design for all devices
- Mobile-friendly navigation with hamburger menu
- Booking modal for appointments
- Newsletter subscription form
- Smooth scrolling
- WhatsApp integration for direct contact
- Modern, clean aesthetic design

## Usage

Simply open `index.html` in a web browser to view the website.
